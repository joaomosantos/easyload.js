EasyLoad.js

Is a lightweight and simple plugin based on jQuery.

Images are loaded as they enter the viewport, simply and easily.

Supports IE7+ and modern browsers and Support jQuery version >= 1.7.x.

Download Github

Getting Started

Package Manager:

bower install --save easyload-js

Usage:

$('.easyload').easyload();

For a simple image load, use the following structure:

<div class="easyload">
 <img src="" data-src="image_path">
</div>

Use css for transition effect:

.easyload img {
 opacity: 0;
 max-width: 100%;
 max-height: 100%;
}

.easyload img.loaded {
 transition: opacity 1s;
}
